How to install Effects

After downloading, extract the null-jsfx folder to your Reaper Effects directory:

Windows
C:/Users/[Username]/AppData/Roaming/REAPER/Effects/
macOS
~/Library/Application Support/REAPER/Effects/
Linux
~/.config/REAPER/Effects/

Restart Reaper or re-scan your plugins folders. NULL JSFX will be available under the JS category in the FX browser.

💡 Tip: To see all plugins organized in the FX explorer, filter by searching "NULL" or create a Smart Folder: Menu "View" → FX Browser → right-click on "Folders" → "Add folder" → "Smart folder" → Filter contains "NULL". They will auto-categorize!

FX Chains

Pre-configured plugin combinations for common workflows.

How to install FX Chains

Copy the .RfxChain files to your Reaper FXChains folder:

Windows
C:/Users/[Username]/AppData/Roaming/REAPER/FXChains/
macOS
~/Library/Application Support/REAPER/FXChains/
Linux
~/.config/REAPER/FXChains/

In Reaper: open the FX window, navigate to the FX Chains category in the left panel. Select your chain — all plugins load automatically in the correct order.

⚠️ Important: For FX Chains to load correctly, the null-jsfx plugins must be installed in the Effects folder as a subfolder (not scattered files). Path example: Effects/null-jsfx/*.jsfx
